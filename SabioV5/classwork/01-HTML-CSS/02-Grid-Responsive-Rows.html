Getting to Know The Bootstrap Grid

The Grid System is based on a 12 Column Row
Class names that take the following format col-{breakpoint}-{columns} are used to determine the number of columns a div should occupy at the given breakpoint and above.
The different breakpoint options are:
  • Bootstrap considers mobile devices in portrait mode the default so there is no class name prefix for it.
  • sm
  • md
  • lg
  • xl
